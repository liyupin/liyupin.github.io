mysql8.0版本忘记root密码解决方法

mysql8.0版本忘记root密码解决方法

一、免密登录mysql

1、停止mysql服务

cmd窗口输入net stop mysql，若为windows下集成环境则直接通过工具停止服务，若为centos6：service mysqld stop,若为centos7：systemctl stop mysqld

配置skip-grant-tables，重启mysql

2、设置免密登录

mysqld –console –skip-grant-tables –shared-memory

【mysqld –skip-grant-tables实测在mysql8.0中已失效】

3、另开一个cmd窗口

登录mysql，password直接回车即可（这边需要另开一个cmd窗口）

二、重新设置mysql密码

1、进入到mysql数据库，设置空密码

2、重启mysql，用空密码进入数据库，修改root密码

（若为windows下集成环境则直接通过工具重启服务，若为centos6：service mysqld restart，若为centos7：

3、flush privileges;//刷新权限列表

4、重新登陆，新密码生效。**

三、在mysql8.0中的无效方法

无效方法一：
修改成功，但是不符合mysql密码加密要求
无效方法二：
update user set authentication_string=password(‘23456’) where user=‘root’;
password()加密函数在mysql8.0版本中不支持，已做调整。
